

### ✅ Prerequisites

* **Java 17+**
* **Maven** or **Gradle**
* **MySQL/PostgreSQL**
* **IDE** (IntelliJ, VS Code)

---

### ⚙️ Setup Steps

1. **Clone or Create Project**
    
2. **Check Dependencies (pom.xml)**
 

3. **Configure `application.properties`**

    
5. **Run the App**

   ```bash
   mvn spring-boot:run
   ```

6. **Test Endpoints**

    * Use **Postman**
    * Or check **Springdoc Swagger UI**: 

      Access: `http://localhost:8080/swagger-ui.html`
